# STRDR App Store Screenshots Guide

## Screenshot Specifications

### iPhone Screenshots (6.7" display - iPhone 14 Pro Max)
- **Resolution**: 1290 x 2796 pixels
- **Format**: PNG
- **Color Space**: sRGB
- **Orientation**: Portrait

### Android Screenshots (Phone)
- **Resolution**: 1080 x 1920 pixels (minimum)
- **Format**: PNG or JPEG
- **Aspect Ratio**: 16:9 or 9:16

## Screenshot Mockup Process

### Tools Needed
- **Figma** or **Sketch** for design
- **iPhone/Android mockup templates**
- **App screenshots** from actual device
- **Marketing overlay graphics**

### Step-by-Step Process
1. **Capture Screenshots**: Take actual app screenshots on target devices
2. **Import to Design Tool**: Place screenshots in device mockups
3. **Add Marketing Overlays**: Apply text overlays with brand styling
4. **Brand Consistency**: Ensure STRDR visual identity throughout
5. **Export**: Save as PNG at required resolutions
6. **Quality Check**: Verify text readability and brand consistency

## App Preview Video Storyboard

### 30-Second App Preview
**Frame 1 (0-3s): Logo Animation**
- STRDR logo appears with lightning animation
- Tagline fades in: "CADENCE AND SPEED OPTIMIZER"
- Clean white background

**Frame 2 (3-8s): Mode Selection**
- Quick transition through training modes
- Show Basic → Terrain → Fartlek → Interval → Progressive
- Highlight professional configuration options

**Frame 3 (8-15s): Live Workout**
- Runner using app during actual run
- Show terrain detection in action
- Display real-time cadence adjustments
- Include voice coaching notifications

**Frame 4 (15-22s): Data Analysis**
- FIT file upload animation
- Charts and graphs appearing
- Professional insights and recommendations
- Show compatibility with multiple devices

**Frame 5 (22-27s): Results**
- Workout completion celebration
- Performance improvement metrics
- User satisfaction/success moment

**Frame 6 (27-30s): Call to Action**
- STRDR logo with download prompt
- App Store badges
- "Download Now" text

### Video Style Guide
- **Pace**: Fast, energetic (matching "speed" theme)
- **Music**: Upbeat, motivational running music
- **Transitions**: Quick cuts, lightning-fast animations
- **Colors**: Consistent white/black theme
- **Text**: Bold, uppercase, high contrast

## Quality Checklist

### Before Submission
- [ ] All screenshots at correct resolution
- [ ] Text overlays readable on all devices
- [ ] Brand consistency across all images
- [ ] No UI elements cut off or obscured
- [ ] Marketing messages clear and compelling
- [ ] STRDR logo/branding visible but not overwhelming
- [ ] Screenshots show actual app functionality
- [ ] Color scheme matches app design
- [ ] Text follows brand typography guidelines
- [ ] All images optimized for file size

### App Store Optimization
- [ ] Screenshots tell a clear story
- [ ] First screenshot is most compelling
- [ ] Each screenshot highlights different feature
- [ ] Marketing text addresses user pain points
- [ ] Visual hierarchy guides eye to key features
- [ ] Screenshots work well as thumbnails
- [ ] Consistent with app description and keywords
