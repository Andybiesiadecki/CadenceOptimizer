# STRDR App Icon Design Brief

## Brand Identity
- **App Name**: STRDR
- **Tagline**: Cadence and Speed Optimizer
- **Theme**: Fast, sleek, precision-focused running performance

## Design Concept
**Primary Concept**: Minimalist lightning bolt + running motion

### Icon Elements:
1. **Lightning Bolt** (⚡) - Represents speed and energy
2. **Circular Motion Lines** - Suggests cadence/rhythm
3. **Clean Typography** - "STRDR" in bold, modern font

## Color Scheme
- **Primary**: Pure White (#FFFFFF)
- **Secondary**: Deep Black (#000000)
- **Accent**: Optional subtle gray (#F8F8F8) for depth

## Style Guidelines
- **Minimalist**: Clean, uncluttered design
- **High Contrast**: Strong black and white contrast
- **Scalable**: Works at all sizes (16px to 1024px)
- **Athletic**: Conveys performance and precision

## Icon Variations Needed

### 1. Primary App Icon (1024x1024)
- White background
- Black lightning bolt in center
- Subtle motion lines around edges
- "STRDR" text below or integrated

### 2. Adaptive Icon (Android)
- Foreground: Lightning bolt + motion lines
- Background: Pure white
- Safe area compliance for different shapes

### 3. Alternative Concepts
- **Concept A**: Lightning bolt with circular cadence rings
- **Concept B**: Stylized "S" with speed lines
- **Concept C**: Metronome pendulum with lightning accent

## Required Sizes

### iOS
- **App Icon**: 1024x1024px (PNG, no transparency)
- **Notification Icon**: 20x20px, 40x40px, 60x60px
- **Settings Icon**: 29x29px, 58x58px, 87x87px
- **Spotlight Icon**: 40x40px, 80x80px, 120x120px
- **App Icon**: 60x60px, 120x120px, 180x180px

### Android
- **Launcher Icon**: 512x512px (PNG)
- **Adaptive Icon**: 512x512px foreground + background
- **Notification Icon**: 24x24dp (white/transparent)

### Web
- **Favicon**: 32x32px, 16x16px
- **Web App Icon**: 192x192px, 512x512px

## Technical Requirements
- **iOS**: 1024x1024px PNG (no transparency)
- **Android**: 512x512px + adaptive icon assets
- **Web**: 192x192px, 512x512px PWA icons
- **Formats**: PNG, SVG (vector source)

## Inspiration Keywords
- Speed, precision, rhythm, performance
- Nike Run Club, Strava, Garmin Connect aesthetics
- Minimalist, professional, athletic
- Lightning, motion, cadence, optimization

## Usage Context
The icon should work well:
- On iOS/Android home screens
- In app stores (small thumbnails)
- As splash screen element
- In marketing materials
