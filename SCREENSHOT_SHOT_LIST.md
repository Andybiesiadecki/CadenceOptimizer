# STRDR Screenshot Shot List

Take these on your phone in order. Keep the app in a clean state for each shot.

## Shot 1: Metronome Home (the hero shot)
- Set cadence to 170 SPM
- Mode set to BASIC
- Metronome NOT playing (show the START button)
- This is the first thing people see in the App Store

## Shot 2: Training Modes
- Scroll down to the TRAINING MODE section
- Select FARTLEK so it's highlighted
- Make sure all 5 modes are visible (Basic, Terrain, Fartlek, Interval, Progressive)

## Shot 3: Pre-Workout Check-In
- Switch to Fartlek mode, tap START
- The "How are you feeling?" modal should appear
- Screenshot the modal with all 4 options visible

## Shot 4: Active Workout
- Select a feeling and let the workout start
- Wait a few seconds so the WORKOUT STATUS section shows
- Capture with the phase indicator, progress bar, and STOP button visible

## Shot 5: Race Target Calculator
- Go to the Targets screen
- Set distance to Half Marathon
- Enter a target time (e.g. 1:30:00)
- Tap Calculate so the results show
- Capture with the cadence, pace, and stride length results visible

## Shot 6: Runner Profile
- Go to the Profile setup screen
- Show the first step or a filled-out step
- This shows personalization

---

## Tips
- Turn off Do Not Disturb so the status bar is clean
- Make sure battery is decent (nobody wants to see 12%)
- Use the phone's built-in screenshot (Side button + Volume Up, or Side + Home on iPhone 8)
- Screenshots save to Photos automatically
